Live Event Hooks

Achievements unlock when the Observatory emits custom events. Dispatch to window to sync from any page.

Keyboard: dispatchEvent(new CustomEvent('observatory:keyboardShortcut', { detail: { key: 'konami' } }))
Hidden Pages: dispatchEvent(new CustomEvent('observatory:hiddenPageVisited', { detail: { slug: 'deploy-450-investigation' } }))
Interactive: dispatchEvent(new CustomEvent('anomalyHuntCompleted')) or dispatchEvent(new CustomEvent('observatory:archiveShuffle'))

Already-tracked visitor stats (from visitor-tracking.js) hydrate progress on load. Storage is shared across Observatory pages.
