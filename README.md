# The Automation Observatory 🔍

An interactive world exploring system patterns, automation, and the persistent marks left by observers.

## Overview

Welcome to The Automation Observatory - a sleek command-center style interactive world where visitors can:
- Explore different monitoring stations
- Watch simulated systems with exponential, incremental, and clockwork patterns
- Discover hidden anomalies in the system timeline
- Submit permanent anomaly reports
- See their marks join the growing archive of observations

## Architecture

- **Frontend:** Interactive HTML/CSS/JavaScript
- **Hosting:** GitHub Pages
- **Persistent Marks:** Stored via GitHub Issues + JSON ledger
- **Visual Style:** Data-center command center aesthetic with timeline visualization

## Features

1. **Pattern Observatory** - Watch three system archetypes evolve in real-time
2. **Deploy Timeline** - Interactive timeline showing system evolution with anomalies
3. **Anomaly Detection** - Find and report system deviations
4. **Archive Walls** - View permanent visitor marks and observations
5. **Responsive Design** - Works on desktop and mobile

## How Visitors Leave Permanent Marks

Visitors can submit anomaly reports which are stored permanently:
1. Fill out anomaly observation form
2. Report includes timestamp, pattern type, observation
3. Marks are persisted in a git-backed JSON ledger
4. All marks appear on the archive walls forever
