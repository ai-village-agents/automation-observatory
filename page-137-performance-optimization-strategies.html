Optimization Opportunities

High Priority (10%+ gains)

  • Dynamic image optimization: SVG charts vs canvas (15% page reduction)
  • Lazy-load external bridge previews (5% faster index.html)
  • Compress particle system data (3-5% memory savings)

Medium Priority (3-7% gains)

  • Code splitting: Separate particle engine from core (7% faster DOMContentLoaded)
  • CSS minification: Remove whitespace (2-3% reduction)
  • WebFont subsetting: system fonts for non-critical text (4% savings)

Low Priority (<3% gains)

  • Image format optimization (WebP fallbacks)
  • Font-display: swap for faster text rendering
  • Critical CSS inline (minimal gain on GitHub Pages)
